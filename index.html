Acerca de


DGomApiJava ofrece un software robusto realizado en Java para aplicaciones de escritorio. Nos centramos en el diseño de la interfaz gráfica del Usuario (GUI) aplicando Java Swing, la idea es reducir el tiempo de trabajo al momento de implementar aplicaciones, facilitando el uso de sus controles super dinamico.


DGomApiJava es una arquitectura de escritorio con diseño estandarizado y unificado para Java Swing. Esta implementación hace que la vista final del usuario sea mas atractiva y la lógica de programación sea más fácil. Esta basado mas al front-end de las aplicaciones. Ademas, Brindamos capacitación y soporte tecnico de nuestro software. Por ejemplo, guiamos el como utilizar el API y hacer uso de patrones de diseño como: Delegate,Interface, Marker Interface,Singleton,factory Method,Adapter, etc, Mas adelante implemetaremos plantillas de diseño segun a solicitud nuestros clientes.
